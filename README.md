# Notary Mock Website

This is a GitHub Pages-ready single-page HTML website for a notary business.

## Files
- `index.html` — main website file

## How to publish with GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the root of the repository.
3. In GitHub, go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or `master`)
   - **Folder:** `/root`
5. Save.
6. Wait a minute or two for GitHub Pages to publish.

Your site will be live at a URL like:
`https://yourusername.github.io/repository-name/`

## Things to customize
Search inside `index.html` and replace:
- `Premier Notary Services`
- `[Notary Name]`
- `(555) 555-0123`
- `hello@premiernotary.com`
- service areas and testimonials

## Notes
- The form is visual only right now and does not send messages.
- The page uses remote image URLs from Unsplash for mockup/demo use.
- For a real client site, you may want to replace those with licensed client photos.
